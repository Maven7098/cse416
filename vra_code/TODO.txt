By default the code does not generate plans - the argument for number of plans should be input separately.
The code generates the following:
    District-level box and whisker plots for each racial category
    Ensemble Charts for Democratic/Republican victories
    Minority-Effectiveness vs Minority-Majority Charts
    Box and Whisker for VRA and Non-VRA districts for Minority and White groups
    Minority-Effectiveness Comparison for VRA and Non-VRA districts
    
    The files are generated as html, json, and png.

source new-vra/bin/activate
python3 new_vra_parallel.py IA-Precinct-EI-GeoJSON.json
                            Complete_256_High/NEW-IA-Precinct (Directory / Filename of the output files)
                            4 (Number of Districts of State - IA has 4, GA has 14)
                            256 (Number of Districts to generate)
                            0.75 (Target Effectiveness)
